## 🇧🇷 Versão em Português

# Gerador de Chaves Aleatórias 🔐

Uma aplicação web simples e eficiente desenvolvida em **Python** (Django) para gerar senhas seguras e aleatórias instantaneamente.

### 🔗 Demonstração

Você pode testar a aplicação ao vivo aqui: [ordnavile.pythonanywhere.com](https://www.google.com/search?q=https://ordnavile.pythonanywhere.com/)

### 🚀 Sobre o Projeto

Muitas pessoas utilizam senhas fracas em cadastros rápidos por falta de tempo ou criatividade. Este projeto visa resolver esse problema de segurança, fornecendo chaves robustas que incluem números, letras (maiúsculas e minúsculas) e caracteres especiais.

### ✨ Funcionalidades

  * **Customização:** Escolha o comprimento da senha entre 1 e 74 caracteres.
  * **Segurança:** Gera combinações aleatórias que dificultam ataques de força bruta.
  * **Rapidez:** Interface minimalista focada na utilidade imediata.

### 🛠️ Tecnologias Utilizadas

  * [Python](https://www.python.org/)
  * [Django](https://www.djangoproject.com/) (Framework Web)
  * [HTML/CSS]

### 💻 Como executar o projeto localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/elivandrosantos/geradorchavealeatoria.git
    ```
2.  Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
3.  Inicie o servidor:
    ```bash
    python manage.py runserver
    ```

## 🇺🇸 English Version

# Random Key Generator 🔐

A simple and efficient web application built with **Python** (Django) to generate secure and random passwords instantly.

### 🔗 Live Demo

Check it out here: [ordnavile.pythonanywhere.com](https://www.google.com/search?q=https://ordnavile.pythonanywhere.com/)

### 🚀 About the Project

Many users rely on weak passwords for daily registrations due to time constraints. This project addresses this security gap by providing robust keys that include numbers, letters (uppercase and lowercase), and special characters.

### ✨ Features

  * **Customization:** Choose a password length between 1 and 74 characters.
  * **Security:** Generates random combinations to prevent brute-force attacks.
  * **Fast & Lightweight:** Minimalist interface focused on immediate utility.

### 🛠️ Tech Stack

  * [Python](https://www.python.org/)
  * [Django](https://www.djangoproject.com/) (Web Framework)
  * [HTML/CSS]

### 💻 Local Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/elivandrosantos/geradorchavealeatoria.git
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the server:
    ```bash
    python manage.py runserver
    ```
